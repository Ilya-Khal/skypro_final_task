Git

  • git init - Инициализирует новый Git‑репозиторий в текущей папке. Создаёт скрытую папку .git, где хранится вся история изменений, настройки и указатели веток.
  • git status - Показывает текущее состояние репозитория: какие файлы изменены, какие новые, какие удалены, на какой ветке ты находишься.
  • git add - Добавляет изменения в индекс («staging area») — подготавливает файлы к фиксации. git add file.txt — добавить один файл.
  • git commit - Фиксирует изменения из индекса как новый коммит (точку в истории). Сохраняет снимок состояния файлов и сообщение (комментарий), объясняющее, что сделано. Пример: git commit -m "комментарий"
  • git mv - Перемещает или переименовывает файл с учётом Git. Примеры:

    git mv old_name.txt new_name.txt — переименовать.

    git mv file.txt src/ — переместить в папку src

  • git push - Отправляет локальные коммиты на удалённый репозиторий (например, на GitHub, GitLab).
  • git log - Показывает историю коммитов в обратном хронологическом порядке (сначала самые свежие). Выводит хеш коммита, автора, дату и сообщение.
